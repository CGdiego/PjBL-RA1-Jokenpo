Esse é um projeto de Jokenpô em Python com 5 modalidades (Humano X Humano, Humano X Computador, Computador X Computador, 2v2 e Battle Royale). Ele tem centenas de linhas de códigos, animação em ASCII, títulos bonitos com pyfiglet, música com winsound e muito mais!
